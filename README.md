# bin-ui

一套基于vue2.6的UI组件库

## Docs

[document](https://wangbin3162.gitee.io/bin-ui/)

## install 安装

```bash
npm install bin-ui -S
```

## use 使用

使用静态引入

```
<!-- import stylesheet -->
<link rel="stylesheet" href="bin-ui/styles/index.css">
<!-- import bin-ui -->
<script src="bin-ui/bin-ui.umd.min.js"></script>
```

使用

```vue
<template>
    <b-button>测试按钮</b-button>
</template>
```
